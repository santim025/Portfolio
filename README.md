# Portfolio — Santiago Cañón Morales

Portfolio personal de desarrollador backend. Sitio estático con HTML, CSS y JavaScript.

## Cómo verlo en local

```bash
# Desde la carpeta del proyecto
python -m http.server 8000
```

Abre [http://localhost:8000](http://localhost:8000)

## Estructura

- `index.html` — Página principal
- `styles.css` — Estilos y animaciones
- `script.js` — Carruseles, lightbox, partículas y scroll
- `assets/` — Iconos e imágenes de proyectos
