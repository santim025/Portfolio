App móvil

engly-app

Ver código →

Aplicación móvil para aprender inglés con lecciones cortas, quizzes y seguimiento de progreso que funciona sin conexión.

engly-app · screenshot

El reto

Permitir practicar inglés en cualquier momento, incluso sin internet, manteniendo el progreso sincronizado y una experiencia fluida en dispositivos de gama baja.

La solución

Una app Expo + React Native con almacenamiento local en SQLite para el progreso offline, tipado estricto con TypeScript y una arquitectura por features para escalar el contenido.

Stack

ExpoReact Native TypeScriptSQLite

Características

  • Lecciones interactivas con quizzes
  • Progreso y rachas offline
  • Sincronización al recuperar conexión
  • Modo oscuro y accesibilidad

Rol

Diseño, arquitectura y desarrollo completo.
